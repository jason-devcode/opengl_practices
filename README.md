# OpenGL Practices using GLFW, GLAD and OpenGL Core 3.3

Cada práctica se realiza en un sólo fichero fuente main.cpp para simplificar el desarrollo durante el aprendizaje.


```bash
# compile command using GLAD and GLFW headers, glad source and GLFW 3 library
# all projects use opengl core profile 3.3 to use modern functionalities
g++ main.cpp ~/.gldev/glad/src/glad.c -o main -I ~/.gldev/glad/include -I ~/.gldev/glfw/include -L ~/.gldev/glfw/lib -lglfw3
```
